# Fungal Depths: Aldric Health Progression After Stabilization
# Tests that Aldric's health INCREASES after first silvermoss
# Critical: 80 severity = 10 damage/turn, 5 regen = net -5 HP/turn
# Stabilized: 40 severity = 4 damage/turn, 5 regen = net +1 HP/turn
# This validates he survives indefinitely after first cure

# Stabilize Aldric so he doesn't die during the walkthrough
# He starts at 40 HP and loses 5 net/turn, so boost him to survive
@set actors.npc_aldric.properties.health = 100

# Initial state check
ASSERT actors.player.properties.health == 100

# Part 1: Navigate to cavern_entrance where Aldric is
@goto cavern_entrance

# Check Aldric's initial state
{"type":"command","action":{"verb":"examine","object":"aldric"}}

# Wait several turns to let him deteriorate
{"type":"command","action":{"verb":"look"}}
{"type":"command","action":{"verb":"look"}}
{"type":"command","action":{"verb":"look"}}
{"type":"command","action":{"verb":"look"}}
{"type":"command","action":{"verb":"look"}}

# Check his health - should be lower but not dead
{"type":"command","action":{"verb":"examine","object":"aldric"}}

# Part 2: Stabilize Aldric with first silvermoss

# Go to grotto to get silvermoss (there are 2 there)
{"type":"command","action":{"verb":"go","object":"down"}}
{"type":"command","action":{"verb":"take","object":"silvermoss"}}

# Return to cavern entrance
{"type":"command","action":{"verb":"go","object":"up"}}

# Give first silvermoss to Aldric
{"type":"command","action":{"verb":"give","object":"silvermoss","indirect_object":"aldric"}}

# Part 3: Observe health after stabilization
# With 40 severity: 4 damage - 5 regen = +1 HP/turn
# His health should slowly increase

# Wait several turns and check health progression
{"type":"command","action":{"verb":"look"}}
{"type":"command","action":{"verb":"examine","object":"aldric"}}
{"type":"command","action":{"verb":"look"}}
{"type":"command","action":{"verb":"examine","object":"aldric"}}
{"type":"command","action":{"verb":"look"}}
{"type":"command","action":{"verb":"examine","object":"aldric"}}

# Part 4: Fully cure Aldric with second silvermoss

# Go to grotto for second silvermoss
{"type":"command","action":{"verb":"go","object":"down"}}
{"type":"command","action":{"verb":"take","object":"silvermoss"}}

# Return to cavern entrance
{"type":"command","action":{"verb":"go","object":"up"}}

# Give second dose
{"type":"command","action":{"verb":"give","object":"silvermoss","indirect_object":"aldric"}}

# Part 5: Return to nexus
{"type":"command","action":{"verb":"go","object":"east"}}

# SUCCESS: Aldric health progression validated!
# - Critical state causes health loss (-5 HP/turn)
# - Stabilized state causes health GAIN (+1 HP/turn)
# - Aldric survives indefinitely after first silvermoss
# - Second silvermoss fully cures him
