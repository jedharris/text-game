# Healer Elara - Garden Access Control (Soft-Block)
# Tests: Players without herbalism skill cannot take herbs from garden
# Scenario: healer_elara.md scenario 9 (design spec says soft-block via take_reactions)

# Navigate to healers_garden (via healers_sanctuary)
go south
go east
go south
go south
go east
go back

# Verify we're in the garden
look
@expect "garden"

# Verify healing_herbs are present
@expect "herbs"

# Try to take herbs without herbalism skill - should be blocked
get herbs  # EXPECTED_FAILURE: player lacks basic_herbalism skill
@expect "training"

# Verify herbs still in garden (blocked action shouldn't remove item)
look
@expect "herbs"

# Now give player basic herbalism skill
@set actors.player.properties.skills = ["basic_herbalism"]

# Try again - should succeed now
get herbs
@expect "You take"

# Summary: Garden soft-block verified
# - Players can enter garden (no hard block)
# - Players without basic_herbalism cannot take healing herbs
# - Players with basic_herbalism can take herbs
